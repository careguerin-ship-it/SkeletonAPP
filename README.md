# SkeletonAPP

## Descripción General
**SkeletonAPP** es una aplicación móvil híbrida desarrollada con el framework **Ionic** y el entorno **Angular**, orientada a demostrar el dominio en la creación de interfaces modernas y funcionales mediante el uso de **Angular Material** y componentes nativos de Ionic.  

El propósito de la aplicación es ofrecer una estructura base adaptable para proyectos educativos o comerciales, integrando formularios validados, navegación entre páginas y animaciones que mejoran la experiencia del usuario.  

## Objetivo del Proyecto
El objetivo principal es desarrollar una aplicación móvil creativa, útil y funcional, que incorpore buenas prácticas de diseño y programación. SkeletonAPP demuestra la implementación de múltiples páginas, validaciones de formularios, componentes visuales de Angular Material y animaciones personalizadas, cumpliendo los criterios de evaluación establecidos.

## Tecnologías Utilizadas
- **Ionic Framework** (CLI y Componentes)
- **Angular 17+**
- **Angular Material**
- **TypeScript**
- **HTML5 / SCSS**
- **Node.js / npm**
